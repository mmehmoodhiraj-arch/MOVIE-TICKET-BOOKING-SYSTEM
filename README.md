Movie Ticket Booking System
The Movie Ticket Booking System is a DBMS project that manages movies, theaters, show schedules, seat reservations, bookings, and payments. It enables customers to book tickets, select seats, and make payments while maintaining accurate records through a relational database. The project demonstrates key DBMS concepts such as ERD, normalization, primary/foreign keys, and SQL queries.
